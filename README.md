# كراج نادي الفروسية المركزي - PWA

تم تحويل التطبيق إلى تطبيق ويب تقدمي (PWA) ليتمكن من العمل دون اتصال بالإنترنت ويتم تثبيته على الأجهزة.

## الميزات الجديدة:
- تثبيت التطبيق على الهاتف أو الحاسوب
- عمل دون اتصال بالإنترنت (مع بعض القيود)
- تجربة مستخدم محسنة

## كيفية الاستخدام:
1. شغل خادم محلي للتطبيق (مثل: `python -m http.server 8000` في مجلد الصور)
2. افتح المتصفح على `http://localhost:8000`
3. ستظهر رسالة تثبيت التطبيق
4. اضغط تثبيت لتثبيته كتطبيق منفصل

## الملفات المضافة:
- `manifest.json`: ملف بيان التطبيق
- `sw.js`: عامل الخدمة للتخزين المؤقت
- `icon.svg`: أيقونة التطبيق

## ملاحظات:
- تأكد من وجود اتصال بالإنترنت أول مرة لتحميل الموارد
- بعض الميزات مثل الاتصال بجوجل شيت تتطلب اتصال بالإنترنت
